# Para iniciar o projeto:

- [Portal do projeto](website/index.html)

- [Descrição do projeto](ischoolde.md)

Para ver screenshots do produto, acesse a pasta `screenshots`

Para rodar o projeto execute:

Navegue para './front-end'
  - `cd front-end` 


Inicie o servidor:

  - `yarn`
  - `yarn start`
